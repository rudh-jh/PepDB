# raw/ace 目录说明

本目录用于保存 ACE 抑制肽任务相关的原始数据。

## 子目录建议
- `databases/`：公共数据库原始下载内容
- `literature/search_records/`：检索式、检索结果、筛选记录
- `literature/pdfs/`：文献 PDF
- `literature/supplementary/`：补充材料
- `literature/manual_extraction_raw/`：手工提取表格的原始版本

## 原则
1. 不直接修改原始文件
2. 尽量保留下载时间、来源链接、来源说明
3. 后续清洗与整理结果不要回写到本层
