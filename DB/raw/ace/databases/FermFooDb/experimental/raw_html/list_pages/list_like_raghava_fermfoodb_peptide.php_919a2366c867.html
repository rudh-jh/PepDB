welcome to Peptide Search Module of FermFooDb


This module provides facility to search a given query peptide against FermFooDb. There are two options
(i) Exact peptide: It facilitate searching of identical peptides in database.
(ii) Substring peptide: It is for searching of peptides containing a part of query peptide.
For more information see HELP page.
